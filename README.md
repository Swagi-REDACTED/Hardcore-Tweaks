A comprehensive mod designed to rebalance and enhance the Minecraft Hardcore experience! Fully updated for Minecraft 26.1-26.1.2.

## Description

 

**Hardcore Tweaks** bridges the gap between normal Survival and punishing vanilla Hardcore. Instead of losing your world permanently on a single mistake, this mod introduces configurable "Death Limits" and "Spawn Penalties" that allow you to balance the difficulty exactly how you want it.

 

Whether you want 3 lives before a Game Over, or you want to punish deaths by randomly scattering the player's respawn location thousands of blocks away, Hardcore Tweaks provides full control over your world's stakes through a seamless in-game GUI.

 

## Features

 

* **Death Limits:** Set a custom amount of "lives" you get per Hardcore world. Once you hit the limit, you trigger the true Hardcore Game Over and enter Spectator Mode.

* **Spawn Penalty:** If you die (and haven't hit the limit), you can opt to be randomly teleported within a configurable block radius instead of returning to world spawn. Your bed/respawn anchor can still be respected if configured!

* **Automated Respawn Timer:** Replaces the instant death screen with a cinematic, vertically-centered 5-second countdown timer that automatically drops you back into the action.

* **First-Time Setup:** A clean, intuitive setup screen greets you the first time you enter a new Hardcore world, letting you establish the rules permanently.

* **In-Game Settings Integration:** ModMenu support and a dedicated "Hardcore Settings" button seamlessly integrated into the vanilla pause menu.

* **Death Tracker HUD:** An aesthetic, customizable on-screen overlay tracking your current death count. Features multiple languages, fonts, scaling, and text case options.

## Compatibility &amp; Parity

 

* **Survival Mode Safe:** Playing in a non-hardcore world? The mod automatically detects standard Survival mode and safely grays out all Hardcore-specific settings, preventing crashes while still allowing you to customize the Death Tracker HUD.

* **Smart Spawning Logic:** The Spawn Penalty system utilizes sophisticated raycasting to ensure you safely land on solid ground, trees, or water, fully preventing unfair void drops or spawning mid-air.
